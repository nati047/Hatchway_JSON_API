# Backend Assessment - Blog Posts
A simple JSON API

## Technologies
  - NodeJs  v12.22.5
  - Express v4.17.3
  - apicache 
  - jest and supertest

## setup & run
  -  ```npm install``` to install dependencies
  -  ```node index.js``` to run server
  - ```npx jest``` to run test
